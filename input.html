# Input 输入框

增强el-input，限制输入。

# 基础用法

:::demo

<template>
  <div>
    只能输入数字：
    <ds-input v-model='number' number style="marginTop :8px"></ds-input>
    只能输入字母：
    <ds-input v-model='english' english style="marginTop :8px"></ds-input>
    只能输入汉字：
    <ds-input v-model='chinese' chinese style="marginTop :8px"></ds-input>
  </div>
</template>
<script>
  export default {
    data() {
      return {
        number: 1234567890,
        english: 'a',
        chinese: '中文',
      }
    }
  }
</script>

:::

# 数字输入精度

在限制为数字输入框的情况下，还可以通过decimal属性控制精度。

:::demo

<template>
  <ds-input v-model='number' number decimal='2'></ds-input>
</template>
<script>
  export default {
    data() {
      return {
        number: '123.125',
      }
    }
  }
</script>

:::

# Input Props

参数 说明 类型 可选值 默认值
v-model 绑定值 String - -
number 只能输入数字 Number - -
english 只能输入英文 String - -
chinese 只能输入中文 String - -
decimal 数字精度 String/Number - -
