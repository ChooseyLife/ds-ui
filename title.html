# Title 标题

标题党专用

# 基础用法

提供两种样式

标题可以通过title或者默认插槽传入

样式一：

:::demo

<template>
  <div>
    <ds-title>基础信息</ds-title>
    <div>...</div>
    <ds-title title="其他信息" style='marginTop: 16px'></ds-title>
    <div>...</div>
  </div>
</template>

:::

样式二：

:::demo

<template>
  <ds-title secondary>基础信息</ds-title>
  <div>...</div>
  <ds-title title="其他信息" secondary style='marginTop: 16px'></ds-title>
  <div>...</div>
</template>

:::

# 自定义标题右侧视图

:::demo

<template>
  <ds-title secondary>
    其他信息
    <ds-toolbar slot='right' :tools='tools'></ds-toolbar>
  </ds-title>
</template>
<script>
  export default {
    data() {
      return {
        tools: [
          {
            text: '保存',
            props: {
              type: 'primary',
            },
            on: {
              click: () => {
                console.log('save!');
              }
            }
          }
        ],
      }
    }
  }
</script>

:::

# Title props

参数 说明 类型 可选值 默认值
title 标题 String - ''
secondary 是否使用样式二 Boolean true/false false

# Title Slots

插槽 说明
default 默认插槽
right 自定义标题右侧视图
