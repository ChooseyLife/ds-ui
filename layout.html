# Layout 布局

应用整体布局组件（本组件强依赖于路由系统在官网这里无法实时演示，仅提供效果图）

本组件需遵循一定的路由配置对象约定才能发挥自动生成菜单，面包屑，菜单伸缩等功能。

路由配置对象约定：

设置meta.title，作为菜单名称提供给递归菜单，面包屑生成时使用。

设置meta.icon， 作为菜单缩略图标。

🙌🌰:

// router.js
export const routes = [
  {
    path: '/base',
    name: 'base',
    component: {
      render: h => <router-view></router-view>
    },
    meta: {
      title: '基础组件',
      icon: 'el-icon-setting',
    },
    children: [
      // 嵌套路由不带'/'
      {
        path: 'toolbar',
        name: 'toolbar',
        component: () => import('./views/toolbar'),
        meta: {
          title: '工具栏',
        },
      },
      // 或者嵌套路由带'/'
      // {
      //   path: '/base/form',
      //   name: 'form',
      //   component: () => import('./views/form'),
      //   meta: {
      //     title: '表单',
      //   },
      // },
    ]
  },
  // ...
]

# 基础用法

  • 传入路由配置则自动生成递归菜单，如果嵌套路由的写法为带'/'的全路径，那么需要在layout上配置isFullPath。

  • 顶部功能区因无法预测，通过插槽headerNav自行实现。

  <template>
    <ds-layout v-bind='layoutProps'>
      <div slot="headerNav">
        <span>这里可能有一级菜单</span>
        <span>退出登录</span>
      </div>
    </ds-layout>
  </template>
  <script>
    import { routes as menus } from './router';
    export default {
      data() {
        return {
          layoutProps: {
            menus,
            // isFullPath: true,
          }
        }
      },
    }
  </script>

# 面包屑

配置showBreadcrumb并且传入breadcrumbList面包屑配置对象，即激活面包屑功能。

TIP

breadcrumbList路由全路径通常通过监听路由从route.matched中获取

🙌🌰:


<script>
export default {
  watch: {
    $route: {
      handler(route) {
        if (route) {
          // TODO 可能还有首页、404时的处理
          const breadcrumb = (route.matched || []).filter(item => item.meta && item.meta.title);
          this.breadcrumbList = breadcrumb.map(item => ({ path: item.path, title: item.meta.title }));
        }
      },
      immediate: true,
    },
  },
};
</script>

<template>
  <ds-layout
    :menus="routes"
    :breadcrumbList="breadcrumbList"
    showBreadcrumb
  >
    <div slot="headerNav">
      <span>这里可能有一级菜单</span>
      <span>退出登录</span>
    </div>
  </ds-layout>
</template>
<script>
  import { routes } from './router';
  export default {
    data() {
      return {
        routes,
        breadcrumbList: [
          { path:"/home", title:"首页" },
          { path:"/base", title:"基础组件" },
          { path:"/base/toolbar", title:"工具栏" },
        ],
      }
    },
  }
</script>

# 多页签

配置showMultiTab启用多页签功能，同时支持右键菜单。

<template>
  <ds-layout
    :menus="routes"
    showMultiTab
  >
    <div slot="headerNav">
      <span>这里可能有一级菜单</span>
      <span>退出登录</span>
    </div>
  </ds-layout>
</template>
<script>
  import { routes } from './router';
  export default {
    data() {
      return {
        routes,
      }
    },
  }
</script>

# Layout props

参数 说明 类型 可选值 默认值
headerHeight header的高度 Number - 64
asideWidth 菜单的宽度 Number - 240
logoBackgroundColor 左上角logo区域背景颜色 String - #002140
defaultCollapse 菜单默认展开 Boolean true/false false
triggerClass 菜单展开时字体图标 String - el-icon-s-fold
collapsedTriggerClass 菜单收缩时字体图标 String - el-icon-s-unfold
menus 约定路由配置 Array - -
isFullPath 嵌套路由是否为带'/'的全路径 Boolean true/false false
breadcrumbList 面包屑配置对象(breadcrumb-item)的集合, 具体配置见下表 Array - -
showBreadcrumb 是否启用面包屑 Boolean true/false false
showMultiTab 多页签 Boolean true/false false
copyRight 页面最底部标识copyRight Sring - 物联团队
collapsePostion 菜单伸缩按钮位置 String header/aside header
...elMenu.$props 侧栏菜单el-menu所有属性 - - -
