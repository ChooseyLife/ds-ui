# Toolbar 工具栏

配置化按钮组, 基于el-button封装

# 基础用法

::: demo

<template>
  <ds-toolbar :tools='tools'></ds-toolbar>
</template>

<script>
export default {
  data() {
    return {
      tools: [
        {
          text: '普通按钮',
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
        {
          text: '增加el-button配置',
          props: {
            type: 'primary',
            round: true,
          },
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
        {
          text: '我被隐藏了',
          props: {
            hidden: true,
          },
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
      ]
    }
  },
}
</script>

:::

# 左右对齐

::: demo

<template>
  <ds-title>整体配置</ds-title>
  <ds-toolbar :tools='wholeTools' align='right'></ds-toolbar>
  <ds-title>单独配置</ds-title>
  <ds-toolbar :tools='tools'></ds-toolbar>
</template>

<script>
export default {
  data() {
    return {
      wholeTools: [
        {
          text: '按钮A',
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
        {
          text: '按钮B',
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
      ],
      tools: [
        {
          text: '左对齐',
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
        {
          text: '右对齐',
          props: {
            align: 'right',
          },
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
      ]
    }
  },
}
</script>

:::

# 按钮配置支持函数写法

为了实现按钮更复杂的动态改变

::: demo

<template>
  <ds-toolbar :tools='tools'></ds-toolbar>
</template>

<script>
export default {
  data() {
    return {
      tools: [
        vm => {
          console.log(vm);
          return {
            text: '函数式写法',
            on: {
              click: (vm, ev) => console.log(vm, ev)
            }
          }
        },
      ]
    }
  },
}
</script>

:::

# 支持添加tooltip配置

按钮文本需要辅助说明时可以考虑加上, 如果你用上该配置，说明该按钮功能设计得不够合理 ::: demo

<template>
  <ds-toolbar :tools='tools'></ds-toolbar>
</template>

<script>
export default {
  data() {
    return {
      tools: [
        {
          text: 'tooltip',
          tooltip: {
            content: '这个按钮的功能有点复杂',
          },
          on: {
            click: (vm, ev) => console.log(vm, ev)
          }
        },
      ]
    }
  },
}
</script>

:::

# 自定义渲染

::: demo

<template>
  <ds-toolbar :tools='tools'></ds-toolbar>
</template>

<script>
export default {
  data() {
    return {
      tools: [
        {
          type: 'custom',
          render: (h, vm) => {
            return (
              <div class='flex'>
                <el-button type='primary' onClick={ev => console.log(vm, ev)}>自定义渲染按钮</el-button>
                <el-tooltip content='温馨提示'>
                  <i class='el-icon-question' style={{ marginLeft: '8px'}}></i>
                </el-tooltip>
              </div>
            )
          }
        }
      ]
    }
  },
}
</script>
<style>
  .flex {
    display: flex;
    align-items: center;
  }
</style>

:::

# Toolbar props

参数 说明 类型 可选值 默认值
align 对齐方式 String left/right left
context 指定回传实例 Object - 最近父vue实例
tools 按钮配置对象(item-options)的集合, 具体配置见下表 Array - []

# Item-Options

参数 说明 类型 可选值 默认值
text 按钮名称 String - -
tooltip el-tooltip组件属性 Object - -
hidden 是否隐藏 Boolean true/false false
render 自定义jsx动态渲染 Function - -
itemType 渲染类型,默认button,自定义custom时需要配合render函数使用 String button/custom button
props el-button所有属性 Object - -
on el-button事件处理函数集合，会直接绑定到el-button，在该组件上只有click事件会响应 Object click -
