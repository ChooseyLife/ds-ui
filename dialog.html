# Dialog 对话框

主要在el-dialog的footer插槽上封装了确认confirm/取消cancel功能。

并且在一些常见场景监听confirm事件时会提供默认数据返回(不一定用得上，各取所需)：

默认插槽为ds-form, el-form, el-table, ds-table, 或者form与table的组合。

# 基础用法

监听confirm事件能接收到两个参数：常见场景默认返回的数据集data和done函数。

data中包含了content（返回数据的实例），data（返回数据），reset（重置form表单）。

# data

  • 如果dialog里包含table组件，会返回table的selection数据。
  • 没有则返回form组件model数据。
  • 这两种组件都没有则代表非常见场景，返回null。

# done

手动控制dialog的关闭。

::: demo

<template>
  <ds-container>
    <ds-dialog v-model='visibleDsForm' @confirm='handlerDsFormConfirm'>
      <div slot='title'>ds-form弹出框</div>
      <ds-form v-bind="formProps" :itemConfig='itemConfig'></ds-form>
    </ds-dialog>
    
    <ds-toolbar :tools='tools'></ds-toolbar>
  </ds-container>
</template>

<script>
export default {
  name: 'dialog-test',

  data() {
    return {
      visibleDsForm: false,
      tools: [
        {
          text: '打开ds-form-dialog',
          on: {
            click: (vm) => {
              this.visibleDsForm = true;
            }
          }
        },
      ],
      formProps: {
        col: 3,
        model: {
          a: '',
          b: '',
          c: '',
        },
        rules: {
          a: [{ required: true, message: '请输入', trigger: 'change' }],
          b: [{ required: true, message: '请输入', trigger: 'change' }],
          c: [{ required: true, message: '请输入', trigger: 'change' }],
        }
      },
    }
  },

  computed: {
    itemConfig() {
      return [
        {
          label: '输入框a',
          prop: 'a',
          type: 'input',
        },
        {
          label: '输入框b',
          prop: 'b',
          type: 'input',
        },
        {
          label: '输入框c',
          prop: 'c',
          type: 'input',
        },
      ]
    },
  },

  methods: {
    handlerDsFormConfirm({ content, data, reset }, done) {
      content.validate(valid => {
        if (valid) {
          setTimeout(() => {
            reset();
          	done();
          }, 200);
        }
      });
    },
  }
}
</script>

:::

# api调用

通常会使得代码更加清晰简洁，无需再关注显隐控制逻辑等。

::: demo

<template>
  <ds-container>

          }, {
            date: '2016-05-03',
            name: '王小虎',

# Dialog props

参数 说明 类型 可选值 默认值
v-model 显隐绑定值 Boolean true/false -
size 弹出框大小 String huge/large/medium/small/mini medium
title 标题 String - -
defaultToolbar 是否显示底部按钮 Boolean true/false true
contentHeight 弹出框默认插槽的高度 String - -
toolbar 弹出框底部按钮集合，参考toolbar配置 Array - -

# Dialog Event

事件名称 说明 回调参数
confirm 确认时触发，常见场景会抛出默认返回数据, 同时抛出done函数手动控制dialog的关闭。 （{vm，data，reset[Function]}，done[Function]）
cancel 取消时触发 -

# render props（api调用）

参数 说明 类型 可选值 默认值
props 继承el-dialog所有属性 Object - -
on 监听el-dialog默认事件 和 dsDialog事件（确认confirm/取消cancel） Object - -
size 弹出框大小 String huge/large/medium/small/mini medium
title 标题 String - -
render 自定义默认插槽 Function h => () -
renderTitle 自定义头部插槽 Function h => () -
renderFooter 自定义底部插槽 Function h => () -
